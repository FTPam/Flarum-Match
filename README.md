A [Flarum](http://flarum.org) extension.   
西南交通大学拼车匹配。

## Installation

Install with composer:

```sh
composer require tpam/flarum-match:"*"
```

## Updating

```sh
composer update tpam/flarum-match:"*"
php flarum migrate
php flarum cache:clear
```
